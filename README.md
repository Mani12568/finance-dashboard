# 💰 Finance Dashboard UI

## 📌 Overview
This is a simple and interactive Finance Dashboard built using React.  
It allows users to track financial activity, view summaries, and analyze spending patterns.

---

## 🚀 Features

### ✅ Dashboard Overview
- Total Income, Expense, and Balance summary
- Clean and responsive UI

### ✅ Transactions Section
- View transactions list
- Search by category
- Filter by Income / Expense

### ✅ Role-Based UI
- Viewer: Can only view data
- Admin: Can add transactions

### ✅ Add Transaction
- Add income or expense
- Updates UI instantly

### ✅ Data Visualization
- Pie Chart: Expense breakdown
- Line Chart: Transaction trend

---

## 🛠️ Tech Stack
- React (Vite)
- Tailwind CSS (CDN)
- Recharts

---
